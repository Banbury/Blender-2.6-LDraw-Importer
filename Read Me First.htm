System Requirements

  • Blender 2.63 or higher
  • ldraw_import.py (included with this download)
  • LDraw System of Tools
  • MLCad, LeoCAD, or Bricksmith (all optional)

Installation

Windows, OS X, Linux

  • Open Blender, and click File > User Preferences > Addons > Install from File...
  • Browse for ldraw_import.py, and open it.
  • Find the script name in the list of Addons (search for LDraw if needed), and tick the check mark to activate it.
  • If you want to load the script at all times, click the Save as Default button.

Setting the LDraw Folder Path

  • Open Blender, and click File > Import > LDraw Model (.dat/.ldr)
  • In the bottom left corner of your screen, under the Import LDraw Model header, there is a small field with the title LDraw Path. Modify the field with the proper folder path to your LDraw System of Tools installation path.

That's it! You have successfully installed the Blender 2.6 LDraw Importer script.

Known Bugs


  1. If a brick cannot be found, entire import stops.
  2. Bricksmith LDR files do not properly import, due to different layout than normal LDR files.
  3. Blender crashes if user imports multiple models and available RAM is too low to continue. Can never be fixed, due to various computers with different amounds of RAM.
  4. Alpha colors are not supported.
  5. Find a way to set the LDraw Brick Library path in Blender itself.
  6. Rare bug where Blender 2.63 will not import any brick colors after already importing a model; may have already been fixed.
  7. Rare bug where Blender 2.63 will crash upon importing a model; may have already been fixed.

Planned Features


  1. Conform to standard Blender script API to be considered for official Blender adoption. See this page for more. (All versions)
  2. Do not stop entire import process if brick cannot be found, rather skip the brick and continue with import (V0.8 Beta 2)
  3. Add the ability to properly import Bricksmith LDR models (V0.8 Final)
  4. Adapt the ability to import transparent and metallic colors from Spencer Alves's Blender 2.6 LDraw Importer on GitHub (V0.9)
  5. Perhaps create an EXE installer for Windows that would automatically install the script (V1.0+)
